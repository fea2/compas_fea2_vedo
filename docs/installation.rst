********************************************************************************
Installation
********************************************************************************

Stable
======

Stable releases are available on PyPI and can be installed with pip.

.. code-block:: bash

    pip install compas_fea2_vedo


Latest
======

The latest version can be installed from local source.

.. code-block:: bash

    git clone https://github.com/fea2/compas_fea2_vedo.git
    cd compas_tna
    pip install -e .


Development
===========

To install `compas_fea2_vedo` for development, install from local source with the "dev" requirements.

.. code-block:: bash

    git clone https://github.com/fea2/compas_fea2_vedo.git
    cd compas_tna
    pip install -e ".[dev]"
